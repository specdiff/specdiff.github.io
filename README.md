# specdiff.github.io

GitHub Pages project site for:

- **SpecDiff** — arXiv:2408.05636
- **SpecDiff-2** — arXiv:2511.00606

## Local preview

From the repo root:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Customize

- Main page: `index.html`
- Styling: `assets/css/style.css`
- Lightweight JS (nav + BibTeX copy): `assets/js/main.js`
- Diagrams/icons: `assets/img/`
